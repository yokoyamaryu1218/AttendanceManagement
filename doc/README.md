## 設計書の説明

- **01.要件設計.xlsx**
    - 本システムの概念図と機能一覧を記載しております。

- **02.画面詳細.xlsx**
    - 出退勤画面と管理画面の各画面詳細を記載しております。

- **03.テーブル定義書.xlsx**
    - テーブル定義とE-R図を記載しております。

- **04.試験項目表.xlsx**
    - プログラミング、AWSへデプロイ後に行ったテストの項目内容、結果を記載しております。

- **05.ログイン用アカウント情報.xlsx**
    - 本システムのログイン用アカウント/社員と上司の紐付き/権限などを記載しております。
